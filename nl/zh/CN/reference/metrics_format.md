# 通用格式
{: #metrics_format}

在 Grafana 中定义的用于监视 {{site.data.keyword.Bluemix}} 服务的查询必须符合以下格式：
{:shortdesc}

```
{Source}.{Cloud Type}.{Service Name}.{Region}.[service fields].{Metric type}.{Metric Subtype}.[Functions]
```
{: codeblock}

其中，[service fields] 表示服务或 CF 应用程序的度量值查询的具体字段。

	<th>描述</th>

  </tr>
  <tr>
